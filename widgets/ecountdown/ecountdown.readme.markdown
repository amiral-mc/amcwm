This widget is just a countdown that show hours, minutes and seconds. At the end, redirect the user in another page.

##Requirements

Yii 1.1 or above

##Usage

~~~
[php]
$this->widget('ext.ecountdown.ECountDown', array(
    'seconds' => 4
));
~~~
